# Amazon Sales Analysis Dashboard (Power BI)

## Project Overview

This project presents an **interactive Power BI dashboard** designed to analyze Amazon product sales performance. The dashboard provides insights into sales trends, product performance, and customer engagement using various visualizations. It helps users quickly understand key business metrics and make data-driven decisions.

## Objectives

The main objective of this project is to analyze **Year-To-Date (YTD) sales data** and identify trends, top-performing products, and category-wise sales distribution.

## Key Features

* **YTD Sales by Month (Line Chart)**
  Displays monthly sales trends to identify seasonal patterns and growth trends throughout the year.

* **YTD Sales by Week (Column Chart)**
  Shows weekly sales performance to highlight short-term fluctuations and sales spikes.

* **Sales by Product Category (Heat Map / Table)**
  Provides an overview of sales across different product categories to understand which categories generate the most revenue.

* **Top 5 Products by YTD Sales (Bar Chart)**
  Highlights the top-performing products based on total sales.

* **Top 5 Products by YTD Reviews (Bar Chart)**
  Identifies the most reviewed products to understand customer interest and popularity.

## Key Metrics Displayed

* **YTD Sales:** Total sales generated during the year.
* **QTD Sales:** Sales performance for the current quarter.
* **YTD Products Sold:** Total number of products sold.
* **YTD Reviews:** Total customer reviews received.

## Filters

The dashboard includes interactive filters for:

* **Product Category**
* **Quarter**

These filters allow users to dynamically explore sales data across different segments.

## Tools & Technologies Used

* **Power BI** – Data visualization and dashboard creation
* **Data Modeling** – Structuring and managing sales data
* **DAX (Data Analysis Expressions)** – Creating calculated measures and metrics

## Insights Provided

* Identify seasonal sales trends
* Track weekly sales performance
* Analyze category-wise revenue contribution
* Discover top-selling and most popular products

## Conclusion

This dashboard enables businesses to monitor sales performance, analyze customer engagement, and make strategic decisions using clear and interactive visualizations.
